# Terraform infra tutorial

## 1. Inititalize Terrafrom files:

```sh
terraform init
```

## 2. Validate Terraform files
```sh
terraform fmt
```
## 3. Apply the latest changes to AWS
```sh
terraform apply
```
## 4. Destroys the EC2 Instance after apply
```sh
terraform destroy
```
